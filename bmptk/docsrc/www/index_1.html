bmptk : Bare Metal Programming Toolkit

Bmptk is a library and make-based build system for C++ applications on bare-metal (no OS) 32-bit systems. Current targets are the Nintendo DS and (for testing) Windows.

You can download the bleeding edge code from google code, or directly using 

   git clone https://code.google.com/p/bmptk
or you can download the latest release as zip file. The is in the zip file, or can be acessed online.

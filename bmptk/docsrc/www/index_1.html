bmptk : Bare Metal Programming Toolkit

Summary

Bmptk is a free library and make-based build system for C++ applications on bare-metal (no OS) 32-bit systems. Current targets are the Nintendo DS and (for testing) Windows.

Downloads

You can download the bleeding edge code from google code, or get a git repository: 
   git clone https://code.google.com/p/bmptk
or you can download the latest release as zip file. The documentation is in the zip file, or you can read it online.

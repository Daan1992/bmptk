bmptk : Bare Metal Programming Toolkit

Summary

Bmptk is a free minimalist make-based cross-development toolkit for developing bare-metal micro-controller applications on Windows or Linux, in assembler, C, or C++, using GCC toolchains.

Downloads

Bmptk is very much 'work in progress'. You can download the bleeding edge code from google code, or get it as a git repository: 
   git clone https://code.google.com/p/bmptk
or you can download the latest release as zip file. The manual is in the zip or can be downloaded separately as word or pdf.
